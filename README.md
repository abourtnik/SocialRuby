## Prerequisites

Be sure to have **Ruby >= 5.6** , Ruby on Rails and **SQLIte >= 5.6** installed on your computer/server

## Installation

### Clone project and install dependencies

```shell
git clone https://github.com/Templatebook/templatebook.git
cd SocialRuby
bundle install
```
